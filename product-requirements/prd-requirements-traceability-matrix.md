## 📐 模块详细设计

### 模块 1：文件上传模块

**功能描述**: 支持单文件和批量文件上传

**接口设计**:
```python
POST /api/v1/upload
- 输入：file (multipart/form-data)
- 输出：{"task_id": "xxx", "status": "uploaded"}

POST /api/v1/upload/batch
- 输入：files[] (multipart/form-data)
- 输出：{"task_ids": ["xxx", "yyy"], "count": 2}
```

**处理流程**:
```
1. 接收文件 → 2. 验证格式 → 3. 保存到临时目录 → 4. 生成 task_id → 5. 返回结果
```

**验收标准**:
- [ ] 支持拖拽上传
- [ ] 支持多文件选择
- [ ] 文件大小限制 50MB
- [ ] 格式验证（仅 PDF）
- [ ] 上传进度显示

---

### 模块 2：OCR 处理模块

**功能描述**: 执行 OCR 识别，生成可搜索 PDF

**接口设计**:
```python
POST /api/v1/process/{task_id}
- 输入：task_id (path)
- 输出：{"status": "processing", "eta_seconds": 120}

GET /api/v1/progress/{task_id}
- 输入：task_id (path)
- 输出：{"progress": 45.5, "stage": "ocr", "eta_seconds": 65}

POST /api/v1/cancel/{task_id}
- 输入：task_id (path)
- 输出：{"status": "cancelled"}
```

**处理流程**:
```
1. 读取 PDF → 2. 分析页数 → 3. 图像预处理 → 4. OCR 识别 → 
5. 生成文本层 → 6. 合成 PDF → 7. 优化压缩 → 8. 保存到输出目录
```

**验收标准**:
- [ ] 中文识别率>95%
- [ ] 英文识别率>98%
- [ ] 10 页 PDF 处理时间<2 分钟
- [ ] 支持取消任务
- [ ] 实时进度更新

---

### 模块 3：任务管理模块

**功能描述**: 管理任务队列和状态

**数据结构**:
```python
Task = {
    "task_id": "str (8 位)",
    "user_id": "str",
    "status": "pending|processing|completed|failed|cancelled",
    "stage": "uploaded|analyzing|ocr|optimizing|completed",
    "input_file": "str",
    "output_file": "str",
    "total_pages": "int",
    "processed_pages": "int",
    "progress": "float (0-100)",
    "created_at": "datetime",
    "started_at": "datetime",
    "completed_at": "datetime",
    "eta_seconds": "float"
}
```

**验收标准**:
- [ ] 任务状态准确
- [ ] 支持并发处理（5 个任务）
- [ ] 任务超时处理（30 分钟）
- [ ] 失败重试机制

---

### 模块 4：用户认证模块

**功能描述**: 用户注册、登录、权限管理

**验收标准**:

- [ ] JWT Token 认证
- [ ] Token 过期刷新
- [ ] 权限控制（管理员/普通用户）

### 模块 5：审计日志模块

**功能描述**: 记录所有操作日志，满足合规要求

**日志内容**:
```json
{
    "log_id": "str",
    "user_id": "str",
    "action": "upload|process|download|delete|login|logout",
    "resource": "task_id|file_id",
    "timestamp": "datetime",
    "ip_address": "str",
    "user_agent": "str",
    "result": "success|failure",
    "details": "dict"
}
```

**验收标准**:
- [ ] 日志不可篡改
- [ ] 日志保留 180 天
- [ ] 支持日志查询和导出
- [ ] 符合 21 CFR Part 11 要求
