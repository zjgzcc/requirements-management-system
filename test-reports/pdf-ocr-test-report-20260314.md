## 7. 修复建议

### 7.1 紧急修复

1. **更新 docker-compose.yml 添加必要依赖**
```yaml
command: >
  bash -c "apt-get update && apt-get install -y ghostscript unpaper pngquant tesseract-ocr tesseract-ocr-chi-sim &&
           pip install --no-cache-dir ocrmypdf fastapi uvicorn python-multipart aiofiles &&
           python -m uvicorn main:app --host 0.0.0.0 --port 8000"
```

2. **添加 Nginx 反向代理配置**
```nginx
location /ocr {
    proxy_pass http://api:8000/ocr;
}
location /status {
    proxy_pass http://api:8000/status;
}
location /download {
    proxy_pass http://api:8000/download;
}
location /files {
    proxy_pass http://api:8000/files;
}
```
