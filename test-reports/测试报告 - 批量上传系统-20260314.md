### Bug 详细说明

#### BUG-001: pdfinfo 缺失
- **现象:** 所有测试中出现 "获取页数失败" 警告
- **影响:** 无法准确获取 PDF 页数，影响进度估算
- **修复:** 安装 poppler-utils: `apt-get install poppler-utils`

#### BUG-002: ocrmypdf 缺失
- **现象:** 处理任务时返回 500 错误
- **影响:** 核心 OCR 功能不可用
- **修复:** 安装 ocrmypdf: `pip install ocrmypdf`

#### BUG-003: Python 版本兼容性
- **现象:** `subprocess.run(capture_output=...)` 在 Python 3.6 不支持
- **影响:** 页数获取功能失效
- **修复:** 改用 `stdout=subprocess.PIPE, stderr=subprocess.PIPE`

#### BUG-004: 资源泄漏
- **现象:** `ResourceWarning: unclosed file`
- **影响:** 长时间运行可能耗尽文件句柄
- **修复:** 使用 `with` 语句确保文件正确关闭
