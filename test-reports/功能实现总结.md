### 1. 取消任务功能 ⭐

#### 后端实现 (`api/main.py`)

**新增接口**: `POST /cancel/{task_id}`

```python
@app.post("/cancel/{task_id}")
async def cancel_task(task_id: str):
    """取消正在处理的任务"""
    task = tasks[task_id]
    process = task.get("process")
    
    if process:
        # 温和终止
        process.terminate()
        
        try:
            # 等待 5 秒
            process.wait(timeout=5)
        except subprocess.TimeoutExpired:
            # 强制杀死
            process.kill()
            process.wait(timeout=5)
        
        # 更新状态
        task["status"] = "cancelled"
        task["stage"] = "cancelled"
        task["progress"] = 0
        
        # 清理临时文件
        os.remove(task["input_file"])
        os.remove(task["output_file"])
```

**关键特性**:
- ✅ 优雅终止进程（先 terminate，超时后 kill）
- ✅ 自动清理临时文件
- ✅ 状态更新为 `cancelled`
- ✅ 防止重复取消（状态检查）

#### 前端实现 (`web/index.html`)

**UI 组件**:
- 处理中显示"取消"按钮
- 点击后显示确认对话框
- 取消后隐藏按钮并更新状态

```javascript

    if (!confirm('确定要取消当前任务吗？')) {
        return;

    if (response.ok) {

    }
}

  "eta_formatted": "2 分 0 秒",

```

**进度计算逻辑**:

```python
# 基于页数计算进度百分比
progress = (processed_pages / total_pages) * 100

# 基于历史速度计算 ETA
elapsed = time.time() - start_time
if processed_pages > 0:
    avg_time_per_page = elapsed / processed_pages
    remaining_pages = total_pages - processed_pages
    eta_seconds = remaining_pages * avg_time_per_page
```

**阶段映射**:

async function updateProgress() {
    const data = await fetch(`/progress/${taskId}`).then(r => r.json());
    
    // 更新进度条
    progressFill.style.width = `${data.progress}%`;
    progressFill.textContent = `${Math.round(data.progress)}%`;
    
    // 更新文本
    progressText.textContent = 
        `进度：${Math.round(data.progress)}% (${data.processed_pages}/${data.total_pages} 页)`;
    
    // 更新阶段
    updateStageIndicator(data.stage);
    
    // 更新 ETA
    if (data.eta_seconds > 0) {
        etaTime.textContent = data.eta_formatted;
        processingSpeed.textContent = data.avg_time_per_page;
    }
}
