🔌 API 接口文档

POST /api/navigation/batch-move

批量移动节点到指定的父节点

{ "nodeIds": ["node1", "node2", "node3"], "targetParentId": "targetNodeId" }
POST /api/navigation/batch-delete

批量删除指定的节点

{ "nodeIds": ["node1", "node2", "node3"] }
POST /api/navigation/reorder

重新排序节点（拖拽操作）

{ "draggedNodeId": "sourceNodeId", "targetNodeId": "targetNodeId", "position": "before" // 或 "after", "into" }
POST /api/navigation/duplicate

复制节点

{ "nodeId": "sourceNodeId", "newName": "新节点名称（可选）" }
