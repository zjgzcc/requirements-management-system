## 🔌 API 接口

### GET /api/diff/requirement/:id1/:id2

对比两个需求基线版本

**示例**:
```bash
curl http://localhost:8001/api/diff/requirement/baseline1_id/baseline2_id
```

**响应**:
```json
{
  "success": true,
  "diff": {
    "baseline1": { "id": "...", "version": "V1.0", "name": "..." },
    "baseline2": { "id": "...", "version": "V2.0", "name": "..." },
    "added": [...],
    "removed": [...],
    "modified": [...],
    "unchanged": [...],
    "fieldChanges": [...],
    "stats": {
      "added": 1,
      "removed": 0,
      "modified": 1,
      "unchanged": 1,
      "total": 2
    }
  }
}
```

### GET /api/diff/testcase/:id1/:id2

对比两个用例基线版本

### GET /api/diff/baseline/:id1/:id2

对比两个基线版本

### POST /api/diff/report

生成差异报告

**请求体**:
```json
{
  "mode": "requirement",
  "version1": "baseline_id_1",
  "version2": "baseline_id_2",
  "diff": { ... },
  "format": "word"
}
```
