## 🎨 UI 界面

### 模板管理页面特性

- **现代化设计**: 渐变背景、卡片式布局、响应式设计
- **筛选功能**: 按可见性（全部/公有/私有）、按类型（Excel/Word）
- **模板卡片**: 显示模板名称、类型、可见性、描述、字段预览
- **操作按钮**: 编辑、导出、删除（根据权限显示）
- **模态框**: 创建/编辑模板、字段配置构建器
- **用户反馈**: Toast 消息提示

### 字段配置构建器

支持动态添加/删除字段，每个字段包含：
- 字段名称（显示名称）
- 字段键（数据映射键）
- 字段类型（文本/数字/日期/选项）
- 列宽（Excel 专用）

## 🔒 权限控制

### 访问控制矩阵

| 操作 | 公有模板 | 私有模板 | 系统模板 |
|------|----------|----------|----------|
| 查看 | 所有用户 | 仅创建者 | 所有用户 |
| 编辑 | 创建者/管理员 | 创建者/管理员 | 不可编辑 |
| 删除 | 创建者/管理员 | 创建者/管理员 | 不可删除 |
| 导出 | 所有用户 | 仅创建者 | 所有用户 |

### 安全特性

- ✅ 所有 API 需要 Bearer Token 认证
- ✅ 未授权访问返回 401
- ✅ 越权操作返回 403
- ✅ 系统模板受保护（createdBy: "system"）

## 🚀 使用说明

### 访问模板管理
```
http://localhost:8001/templates.html
```

### 创建模板
1. 登录系统
2. 访问模板管理页面
3. 点击"新建模板"
4. 填写模板信息
5. 配置字段
6. 保存

### 使用模板导出
1. 在模板列表选择模板
2. 点击"导出"按钮
3. 选择数据范围
4. 确认导出
5. 自动下载文件
